Introduction

Styler CSS is a lightweight CSS library designed to simplify the process of building user interfaces. It offers a concise and intuitive syntax that allows developers to write CSS code directly within HTML classes, enhancing readability and maintainability.

Why Use Styler CSS?

  • Simplified Syntax: Styler CSS adopts a natural language-like approach, making it easier to write and understand CSS compared to traditional stylesheets.
  • Improved Readability: Inline styles provide a clear association between the HTML element and its styling, promoting better code organization.
  • Reduced File Size (Potential): The need for separate CSS files might be minimized in smaller projects.
  • Easy Learning Curve: The simplified and intuitive syntax can ease the learning curve for developers new to traditional CSS rules.

Installation

Include Styler CSS in your project by adding the following links to your HTML file, adjusting the paths as necessary:

<link rel="stylesheet" href="path/to/styler.css">
<script src="path/to/styler.js"></script>

Note: Utility scripts like `grid.js` and `flex.js` dynamically generate additional CSS classes, and should be included after your main stylesheet.

Features

  • Grid system: Utility classes for grid containers, columns, row/column spans, gaps, and auto rows.
  • Flex utilities: Flex container and alignment helpers (generated by flex.js).
  • Theme Changing: Easy theme switching functionality via themeChanger.js.
  • Media Queries: Built-in responsive class modifiers (e.g., md:, lg:) via media_query.js.
  • Hover Effects: Pre-built hover animations and effects via hover.js.
  • Custom Styles: Write custom styles directly in HTML classes.

Usage Examples

Basic Styling

Combine utility classes for common styling tasks like background color, text color, and padding.

<div class="bg-gray-200 color-black p-20">
  This div has a gray background, black text, and 20px padding
</div>

Theme Support

The dark mode is activated by the .dark class on the <body> tag, which is managed by the theme changer button.

<body class="dark">
  <!-- Content is automatically styled -->
</body>

Grid and Flex Example

A simple responsive header using grid (for columns) and flex (for alignment) from the core scripts.

<header class="grid grid-cols-12 gap-4">
  <div class="col-span-3">Logo</div>
  <nav class="col-span-9 flex justify-end items-center gap-2">
    <a>Home</a>
    <a>About</a>
  </nav>
</header>

Animations and Effects

Styler CSS includes utility classes for common animations and hover effects. These animations help in creating engaging user interfaces for the web.

Spin Animations (from style.css)

Use .spin-right or .spin-left to create continuous rotation. Add .spin-fast or .spin-slow for speed control.

<i class="spin-right">⟳</i>
<i class="spin-left spin-fast">⟲</i>
⟳ ⟲

Movement Animation (from styler.css)

The .move class applies a complex, keyframed movement across the container. Hover over the box to pause the animation.

<div class="move move-box">Moving Box</div>
Moving Box

Hover Effects (from hover.js)

Apply utility classes like hover-scale for interactive elements.

<button class="hover-scale bg-gray-300 p-10 rounded-5">Hover to scale</button>
